# Quick Start

## Fastest Way to Test

### Chrome/Edge/Brave:
1. Press `Ctrl+Shift+M` to open File Explorer
2. Navigate to: `C:\Users\Today\Documents\Code\Web - Extensions\RemoveRblxPlus`
3. Copy the full path
4. In Chrome, go to: `chrome://extensions/`
5. Turn on **Developer mode** (top-right toggle)
6. Click **Load unpacked**
7. Paste the folder path
8. Go to `roblox.com` - the Roblox Plus button should be gone!

### Firefox:
1. Open Firefox and go to: `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Select `manifest.json` from this folder
4. Go to `roblox.com` - the Roblox Plus button should be gone!

That's it! No configuration needed.
